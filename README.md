A user-friendly Java Swing desktop application to manage your personal expenses.

## Features

- Add expenses with category, amount, date, and note
- View all expenses in a sortable table
- See the total expense
- Data stored in local CSV file

## Running

1. Compile all Java files:

    ```bash
    javac src/*.java
    ```

2. Run the application:

    ```bash
    java src.SmartExpenseUI
    ```
